Vowel consonant identification
------------------------------------------------------------------------
A small Python script written after completing the Discovery Piscine Python program.
This project was created independently as practice to reinforce basic programming concepts.

Description
-------------------------------------------------------------------------

The script prompts the user to enter a letter and determines whether it is:

* A vowel
* A consonant
* Not a valid alphabet letter

It then allows the user to repeat the process.

Purpose of doing this mini project
----------------------------------------------------------------------
This project was built to practice:

* User input handling
* Conditional logic (if , elif , else)
* Functions
* Loops (FOR and WHILE loops)
* String normalization
* Basic program flow design
